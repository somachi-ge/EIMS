# EIMS用户手册文件目录

## 说明

此目录用于存放用户手册的静态文件，供用户下载。

## 文件列表

- `EIMS_User_Manual_V1.0.0.pdf` - PDF格式用户手册
- `EIMS_User_Manual_V1.0.0.docx` - Word格式用户手册

## 生成方法

### PDF版本生成

1. 在浏览器中打开帮助中心的"预览手册"功能
2. 使用浏览器的"打印"功能（Ctrl+P）
3. 选择"另存为PDF"
4. 保存到此目录

或使用专业工具：
- 使用 Puppeteer + html-pdf 自动生成
- 使用 wkhtmltopdf 命令行工具

### Word版本生成

1. 复制 `UserManual.vue` 中的文本内容
2. 粘贴到 Word 文档中
3. 调整格式后保存为 `.docx` 文件

## 注意事项

- 手册版本号应与系统版本保持一致
- 每次系统更新后，需同步更新手册内容
- 建议保留历史版本供用户查阅
